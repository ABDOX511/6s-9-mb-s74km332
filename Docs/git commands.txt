## Essential Git Commands Cheat Sheet

### Setup & Configuration
```bash
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```

### Starting a Repository
```bash
git init                     # Initialize a new Git repository
git clone          # Clone an existing repository
```

### Branching & Switching
```bash
git branch                   # List branches
git branch      # Create a new branch
git checkout    # Switch to a branch
git checkout -b  # Create and switch to a new branch
git branch -d   # Delete a local branch
```

### Making Changes
```bash
git status                   # Show changed files
git add                # Stage a file
git add .                    # Stage all changes
git commit -m "Commit message" # Commit staged changes
```

### Updating & Syncing
```bash
git pull origin      # Fetch and merge changes from remote branch
git push origin      # Push local commits to remote branch
git push origin --delete  # Delete remote branch
```

### Viewing History & Logs
```bash
git log                      # Show commit history
git log --oneline            # Show compact commit history
```

### Undoing Changes
```bash
git reset --soft HEAD~1      # Undo last commit, keep changes staged
git reset --hard HEAD~1      # Undo last commit, discard changes
git checkout --        # Discard changes in working directory for a file
git clean -fd                # Remove untracked files and directories
```

### Rebasing & Merging
```bash
git merge            # Merge a branch into current branch
git rebase -i HEAD~n         # Interactive rebase last n commits
```

### Miscellaneous
```bash
git stash                    # Save changes temporarily
git stash pop                # Apply stashed changes
git remote -v                # Show remote repositories
```
